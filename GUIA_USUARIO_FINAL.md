# 🚀 GUIA RÁPIDO - Como Usar o Aplicativo

## Para Usuários Finais (Sem conhecimento técnico)

---

### ✅ PASSO 1: Baixar o Aplicativo

1. Você recebeu um arquivo chamado `Gestor_Financeiro`
2. Clique nele (ou se for um `.zip`, primeiro extraia)

---

### ✅ PASSO 2: Executar o Aplicativo

## 🍎 **No Mac:**
1. Localize o arquivo `Gestor_Financeiro`
2. Dê um **duplo-clique** nele
3. Se aparecer "não pode abrir", faça assim:
   - Clique com **botão direito** (ou Ctrl+Clique)
   - Escolha **"Abrir"**
   - Clique em **"Abrir"** novamente na caixa que aparecer

## 🪟 **No Windows:**
1. Localize o arquivo `Gestor_Financeiro.exe`
2. Dê um **duplo-clique** nele
3. Se aparecer aviso de segurança:
   - Clique em **"Mais informações"**
   - Clique em **"Executar mesmo assim"**

---

### ✅ PASSO 3: Aguardar Abertura

1. Uma janela preta (console) pode aparecer - deixe aberta
2. Aguarde **2-3 segundos**
3. Seu navegador internet abrirá automaticamente

---

### ✅ PASSO 4: Usar o Aplicativo

Pronto! Você já pode:
- ✅ Registrar transações
- ✅ Ver gráficos
- ✅ Acompanhar saldos
- ✅ Gerenciar categorias

---

## ⚠️ Dúvidas Comuns

### **P: O navegador não abriu**
**R:** Abra seu navegador (Chrome, Firefox, Safari) e digite: `http://127.0.0.1:3333`

### **P: Preciso instalar algo?**
**R:** Não! Tudo já está pronto no executável. Apenas execute o arquivo.

### **P: Meus dados são salvos?**
**R:** Sim! Os dados são armazenados automaticamente no seu computador.

### **P: Posso fechar a janela preta?**
**R:** Não! Enquanto estiver usando o aplicativo, deixe aberta. Fechar a janela preta encerra o aplicativo.

### **P: Como desinstalar?**
**R:** Basta **deletar o arquivo**. Nada é instalado no seu computador.

---

## 🆘 Se der Problema

**Sua janela preta está vazia ou com erro?**

Tente:
1. Fechar tudo
2. Baixar a última versão do aplicativo novamente
3. Se o problema persistir, contate o desenvolvedor

---

## 💡 Dicas

- 💾 **Faça backup**: Copie o arquivo `Gestor_Financeiro` para um local seguro
- 🔄 **Nova versão**: Quando houver atualização, basta substituir o arquivo
- 📌 **Atalho desktop**: Arraste o arquivo para o Desktop para acesso rápido
